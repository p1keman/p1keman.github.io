/proc目录

介绍

proc是一个伪文件系统(系统启动时，动态生成的文件)，不占用存储空间，占用内存空间

ls -al看到文件大小基本都是0

当读取文件内容的时候，会去内存中找到数据然后返回

ls看一下

前面那些名字是数字的对应着PID号 也就是进程号

/proc/self

指向当前进程

举个栗子

web服务 使用apache2进程

/proc/self就可以切换到当前进程 指向apache2

然后在调用cwd就切换到了当前的工作目录

/proc/self/cwd/index.php 就可以读到源码

当找不到网站的web根目录的时候，就可以利用该方式，读源码

ls -l 可以查看当前的进程号
